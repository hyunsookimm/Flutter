> 이 프로젝트에서 사용하는 외부 패키지 4가지의 개념, 설치, 적용 방법을 정리합니다.

## 1. path_provider `^2.0.0`

### 개요

| 항목 | 내용 |
|------|------|
| pub.dev | https://pub.dev/packages/path_provider |
| 역할 | 플랫폼별 파일 시스템 경로를 Flutter에서 얻는 유틸리티 |
| 지원 | Android, iOS, macOS, Windows, Linux |

### 왜 필요한가?

Flutter 앱은 플랫폼마다 파일 저장 위치가 다릅니다.

| 플랫폼 | 경로 예시 |
|--------|----------|
| Android | `/data/user/0/com.example.diary_app/app_flutter/` |
| iOS | `/var/mobile/.../Documents/` |
| Windows | `C:\Users\user\Documents\` |

`path_provider`가 이 경로를 **자동으로 반환**해주므로, 플랫폼 분기 코드 없이 하나의 코드로 동작합니다.

### 주요 API

```dart
import 'package:path_provider/path_provider.dart';

// 앱 전용 문서 디렉토리 (가장 많이 사용)
final dir = await getApplicationDocumentsDirectory();

// 임시 파일 디렉토리 (캐시 용도)
final tmp = await getTemporaryDirectory();
```

### 프로젝트 적용 코드

```dart
// lib/services/file_service.dart
Future<String> getDirPath() async {
  final dir = await getApplicationDocumentsDirectory();
  return dir.path;  // 이 경로 아래에 .txt 파일 저장
}

// 사용 예
final file = File('$dirPath/2026-03-26_143022.txt');
await file.writeAsString('내용');
```

### 설치

```yaml
# pubspec.yaml
dependencies:
  path_provider: ^2.0.0
```

## 2. calendar_date_picker2 `^2.0.1`

### 개요

| 항목 | 내용 |
|------|------|
| pub.dev | https://pub.dev/packages/calendar_date_picker2 |
| 역할 | 다이얼로그 형태의 날짜 선택 달력 위젯 |
| 기능 | 단일 선택 / 범위 선택 / 다중 선택 / 커스텀 스타일 |

### 주요 타입

```dart
// 캘린더 타입
CalendarDatePicker2Type.single   // 단일 날짜
CalendarDatePicker2Type.range    // 시작~종료 범위
CalendarDatePicker2Type.multi    // 여러 날짜 선택
```

### 프로젝트 적용 코드

> `write_page.dart` (일기 작성) · `detail_page.dart` (날짜 수정) 양쪽에서 사용

```dart
import 'package:calendar_date_picker2/calendar_date_picker2.dart';

Future<void> _pickDate() async {
  final result = await showCalendarDatePicker2Dialog(
    context: context,
    config: CalendarDatePicker2WithActionButtonsConfig(
      calendarType: CalendarDatePicker2Type.single,  // 단일 선택
      selectedDayHighlightColor: Colors.amber,       // 선택일 강조색
      okButton: const Text('확인', style: TextStyle(color: Colors.amber)),
      cancelButton: const Text('취소'),
    ),
    dialogSize: const Size(325, 400),
    borderRadius: BorderRadius.circular(15),
    value: [_selectedDate],   // 초기 선택값
  );

  // result: List<DateTime?> — null이 아닐 때만 업데이트
  if (result != null && result.isNotEmpty && result.first != null) {
    setState(() => _selectedDate = result.first!);
  }
}
```

### 적용 위치

| 화면 | 기능 |
|------|------|
| `WritePage` | 일기를 쓸 날짜 선택 (기본값: 오늘) |
| `DetailPage` (edit 모드) | 기존 일기의 날짜 변경 |

### 설치

```yaml
dependencies:
  calendar_date_picker2: ^2.0.1
```

## 3. flutter_multi_select_items `^0.4.3`

### 개요

| 항목 | 내용 |
|------|------|
| pub.dev | https://pub.dev/packages/flutter_multi_select_items |
| 역할 | 체크박스 스타일의 다중 선택 UI 컨테이너 |

### ⚠️ 주의 — 이 프로젝트에서의 처리

패키지 설치는 되어 있으나, **실제로는 사용하지 않습니다.**

#### 사용하지 않은 이유

`MultiSelectContainer` 위젯이 내부적으로 `Wrap`을 사용하는데,  
이를 `SingleChildScrollView` + `Expanded` 안에 넣으면 **무한 높이 레이아웃 오류**가 발생:

```
RenderFlex children have non-zero flex but incoming height constraints are unbounded.
```

#### 대안: 직접 구현

```dart
// home_page.dart — Set 기반 수동 다중 선택
Set<String> _selectedPaths = {};

// 탭 시 토글
onTap: () => setState(() {
  if (_selectedPaths.contains(path)) {
    _selectedPaths.remove(path);
  } else {
    _selectedPaths.add(path);
  }
});

// 전체 선택
_selectedPaths = _filteredList.map((e) => e.path).toSet();

// 전체 해제
_selectedPaths.clear();
```

`Set<String>`으로 선택 경로를 관리하면 중복 없이 O(1) 조회·추가·제거가 가능합니다.

### 설치

```yaml
dependencies:
  flutter_multi_select_items: ^0.4.3
```

## 4. table_calendar `^3.2.0`

### 개요

| 항목 | 내용 |
|------|------|
| pub.dev | https://pub.dev/packages/table_calendar |
| 역할 | 월/주/이틀 뷰를 지원하는 완전한 기능의 달력 위젯 |
| 핵심 기능 | 이벤트 마커, 날짜 선택, 커스텀 스타일, 스와이프 페이지 전환 |

### 주요 파라미터

```dart
TableCalendar<T>(
  firstDay: DateTime(2020),    // 달력 시작일
  lastDay: DateTime(2100),     // 달력 종료일
  focusedDay: _focusedDay,     // 현재 표시 중인 달
  selectedDayPredicate: (day) => isSameDay(_selectedDay, day),  // 선택일 판별
  eventLoader: _eventsFor,     // 날짜 → 이벤트 리스트 함수
  onDaySelected: _onDaySelected,   // 날짜 선택 콜백
  onPageChanged: (day) => setState(() => _focusedDay = day),
)
```

### 이벤트 마커 원리

```dart
// 날짜 문자열(yyyy-MM-dd) → DiaryEntry 목록 매핑
Map<String, List<DiaryEntry>> _eventMap = {};

// eventLoader: 달력이 날짜를 렌더링할 때마다 호출
List<DiaryEntry> _eventsFor(DateTime day) {
  final key = '${day.year}-${day.month.toString().padLeft(2, '0')}'
              '-${day.day.toString().padLeft(2, '0')}';
  return _eventMap[key] ?? [];  // 해당 날짜 일기 목록 반환
}
```

- 반환된 리스트 크기만큼 **주황 점(marker)** 이 날짜 아래에 표시됨
- `markersMaxCount: 3` → 최대 3개 점까지만 표시

### 프로젝트 적용 — `calendar_page.dart`

```dart
// 날짜 선택 시 처리
void _onDaySelected(DateTime selectedDay, DateTime focusedDay) {
  setState(() {
    _selectedDay = selectedDay;
    _focusedDay = focusedDay;
  });
  final entries = _eventsFor(selectedDay);
  if (entries.isEmpty) return;
  if (entries.length == 1) {
    _openDetail(entries.first);       // 1개 → 바로 상세보기
  } else {
    _showPickerSheet(entries);        // 2개 이상 → BottomSheet 목록
  }
}
```

### 커스텀 스타일 요약

```dart
calendarStyle: CalendarStyle(
  todayDecoration: BoxDecoration(color: Colors.amber.shade300, shape: BoxShape.circle),
  selectedDecoration: BoxDecoration(color: Colors.amber, shape: BoxShape.circle),
  markerDecoration: BoxDecoration(color: Colors.deepOrange, shape: BoxShape.circle),
),
headerStyle: HeaderStyle(
  formatButtonVisible: false,  // 월/주 전환 버튼 숨김
  titleCentered: true,
),
```

### 설치

```yaml
dependencies:
  table_calendar: ^3.2.0
```
